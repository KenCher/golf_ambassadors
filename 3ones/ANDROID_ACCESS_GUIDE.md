# 📱 Android Access Guide - Golf Ambassadors Event Registration

## 🚀 Quick Access URLs (After Firebase Deployment)

Once deployed to Firebase, you'll have these URLs accessible from any Android phone:

### For Admins:
```
https://your-app.web.app/event-registration-admin.html
```
**Use this to:**
- Create new golf events
- Generate registration links
- View all registrations
- Enter scores for players
- Export registration data

### For Members (Registration):
```
https://your-app.web.app/event-registration.html?event=EVENT_ID
```
**Members use this to:**
- Register for specific events
- Enter name and handicap
- Confirm 1:1:1 contest participation

### For Score Entry:
```
https://your-app.web.app/golf-ambassadors.html
```
**Players use this to:**
- Enter their own scores after the round
- View their scorecard
- Mark 1:1:1 contest wins

### For Viewing Results:
```
https://your-app.web.app/golf-ambassadors-results.html
```
**Everyone can:**
- View live leaderboards
- See Stableford rankings
- Check 1:1:1 contest winners

## 📲 How to Access on Android Phone

### Method 1: Direct Browser Access
1. Open **Chrome** or **Firefox** on your Android phone
2. Type or paste the URL
3. Bookmark it for easy access

### Method 2: Add to Home Screen (Recommended)
1. Open the URL in Chrome
2. Tap the **menu** (⋮) in the top-right
3. Select **"Add to Home screen"**
4. Name it (e.g., "Golf Events")
5. Tap **"Add"**
6. Now you have an app-like icon on your home screen! 📱

### Method 3: WhatsApp Link
1. Admin shares the link via WhatsApp
2. Tap the link to open
3. Register or enter scores directly

## 🎯 Complete Android Workflow

### **ADMIN WORKFLOW:**

**Step 1: Create Event (On Computer or Phone)**
1. Open: `https://your-app.web.app/event-registration-admin.html`
2. Fill in event details
3. Click "Create Event"
4. Copy the registration link

**Step 2: Share Registration Link**
```
🏌️ GOLF EVENT - REGISTER NOW!

Event: Monthly Challenge
📅 Date: April 30, 2026
⛳ Course: Deerfield Golf Course

👉 REGISTER HERE:

```

**Step 3: Monitor Registrations (On Phone)**
1. Open admin portal on your Android
2. Go to "Manage Registrations" tab
3. Select your event
4. View all registered players in real-time

**Step 4: After Event - Enter Scores (On Phone)**
1. Open admin portal
2. Go to "Enter Scores" tab
3. Select event and player
4. Enter scores
5. Save

### **MEMBER WORKFLOW:**

**Step 1: Register (On Android Phone)**
1. Receive WhatsApp message with registration link
2. Tap the link

   - Your name
   - Your handicap
   - Check 1:1:1 contest box (if participating)

**Step 2: Play Golf**
- Enjoy your round
- Keep track of scores
- Try to win 1:1:1 contest!

**Step 3: Enter Scores (On Android Phone)**
1. Receive score entry link from admin
2. Tap the link
3. Select your name
4. Enter scores for each hole
5. Mark 1:1:1 win if applicable
6. Submit

**Step 4: View Results (On Android Phone)**
1. Tap results link
2. See live leaderboard

## 🔧 Troubleshooting on Android

### "Page Not Loading"
**Solution:**
- Check internet connection (WiFi or mobile data)
- Clear browser cache
- Try different browser (Chrome vs Firefox)
- Refresh the page

### "Can't Enter Data"
**Solution:**
- Enable JavaScript in browser settings
- Allow cookies
- Clear browser data
- Try incognito/private mode

### "Registration Link Not Working"
**Solution:**
- Check the full URL was copied
- Verify event ID is correct
- Ask admin for new link
- Try opening in different browser

### "Scores Not Saving"
**Solution:**
- Check internet connection
- Don't close browser immediately after submitting
- Wait for confirmation message
- Contact admin if issue persists

## 📊 Data Storage on Android

### How It Works:
- Data stored in browser's **localStorage**
- Persists across sessions
- Specific to each browser
- Cleared if you clear browser data

### Important Notes:
- ⚠️ Don't clear browser data during active events
- ⚠️ Use same browser for consistency
- ⚠️ Admin should backup data (export CSV)
- ✅ Firebase deployment provides cloud backup
