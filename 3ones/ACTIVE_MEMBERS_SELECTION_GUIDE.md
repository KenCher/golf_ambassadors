# Active Members Selection Guide

## ✅ Feature Already Implemented!

The ability to select members from the imported active members list is **fully functional** in the Event Registration Admin page.

---

## 📋 How to Use Active Members Selection

### **Step 1: Import Active Members**

1. Go to **Event Registration Admin** page
2. Click on **"👥 Manage Registrations"** tab
3. Find the **"📥 Active Members Import"** section at the top
4. Click **"Upload CSV File"** button
5. Select your CSV file with member data
6. Click **"Import Members"**
7. You'll see: "X active members available for quick event entry"

### **Step 2: View Imported Members**

After import, you'll see:
- **Member cards** displayed (showing first 30 members)
- Each card shows: Name and Handicap
- **Dropdown selector** populated with all members

### **Step 3: Select Event**

1. In the **"Select Event"** dropdown
2. Choose the event you want to add players to
3. The registration form will appear below

### **Step 4: Use Active Member**

**Method 1: Using Dropdown (Recommended)**
1. Click the **"-- Choose Imported Active Member --"** dropdown
2. Select a member from the list (shows name and handicap)
3. Click the **"Use Member"** button
4. The form auto-fills with:
   - Player Name
   - Handicap
5. Adjust any details if needed
6. Click **"Add Player"** to register them

**Method 2: Manual Entry**
1. Type name and handicap manually
2. Click **"Add Player"**

## 💡 Tips

### **For Fast Registration:**
1. Import your member list once
2. It stays saved in browser
3. Use dropdown for each player
4. Click "Use Member" → "Add Player"
5. Repeat for all participants

### **CSV Format:**
Your CSV should have columns like:
- `Name` or `First Name` + `Last Name`
- `Handicap` or `HCP` or `Index`

Example:
```csv
Name,Handicap
Joe Sang,5.9
Paul Russo,23.4
James Githinji,9.7
```

### **Managing Members:**
- **Clear All:** Click "Clear Imported Members" button
- **Re-import:** Upload new CSV to replace
- **Persistent:** Data stays until cleared

## 🔧 Troubleshooting

### **Dropdown is Empty**
- Make sure you imported the CSV first
- Check that CSV has Name column
- Try refreshing the page

### **"Use Member" Button Not Working**
- Select a member from dropdown first
- Make sure an event is selected
- Check browser console for errors

### **Members Not Showing**
- Verify CSV format is correct
- Check file uploaded successfully
- Look for import success message
