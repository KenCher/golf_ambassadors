3. **On your Android phone:**

### Method 2: Google Drive

1. **Upload to Google Drive:**
   ```bash
   # The file is at:

   ```
   - Upload this file to your Google Drive

2. **On your Android phone:**
   - Open Google Drive app
   - Find `mobile.html`
   - Tap to open
   - Choose "Open with Chrome"

### Method 3: USB Transfer

1. **Connect phone to Mac via USB**
2. **Enable File Transfer on Android**
3. **Copy the file:**
   ```bash
   # Copy from:
   /Users/kencheru/Golf4us/public/mobile.html
   
   # To your phone's:
   Downloads folder
   ```
4. **On phone:** Open Files app → Downloads → mobile.html → Open with Chrome

## 🎮 Using the App

Once opened in Chrome:

1. **Add to Home Screen** (recommended):
   - Tap Chrome menu (⋮)
   - Select "Add to Home screen"
   - Now it's an app icon on your home screen!

2. **Start Using:**
   - Tap "✏️ Score" tab
   - Select a player
   - Select a hole
   - Enter strokes
   - Submit
   - Check "📊 Board" tab to see results!
