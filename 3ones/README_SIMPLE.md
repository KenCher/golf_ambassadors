# 🏌️ Golf Ambassadors - Quick Start

A simple, mobile-friendly golf tournament management app.

## 🚀 Deploy in 3 Steps

### 1. First Time Setup (One Time Only)
```bash
npm install -g firebase-tools
firebase login
```

### 2. Deploy
```bash
./deploy.sh
```

### 3. Done!
Your app is live at: **https://golf-ambassadors.web.app**

## 📱 Use on Mobile

### iPhone/iPad:
1. Open Safari → https://golf-ambassadors.web.app
2. Tap Share → "Add to Home Screen"
3. App icon appears on home screen

### Android:
1. Open Chrome → https://golf-ambassadors.web.app
2. Tap Menu (⋮) → "Add to Home screen"
3. App icon appears on home screen

## 🎯 Main Features

- **Event Management** - Create and manage golf tournaments
- **Player Registration** - Easy sign-up for events
- **Score Entry** - Enter scores for yourself or your foursome
- **Live Leaderboard** - Real-time results and standings
- **Mobile Optimized** - Works perfectly on phones and tablets

## 📂 Key Files

```
public/
├── golf-ambassadors.html          # Score entry
├── golf-ambassadors-results.html  # Leaderboard
├── event-registration-admin.html  # Event management
└── event-registration.html        # Player sign-up
```

## 🔧 Making Changes

1. Edit files in `public/` folder
2. Test by opening HTML files in browser
3. Deploy: `./deploy.sh`
4. Live immediately!
